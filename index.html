Présentation du groupe

Notre groupe est composé de trois membres, Thibault & Léo Lenoir et Lola Pasquier. Nous sommes des étudiants en MMI ayant réalisé un site web de météo dans le cadre d'une SAE (Situation d'Apprentissage et d'Évaluation).

Présentation du projet

La SAE303 vise à concevoir et à mettre en place une application Web de météo centrée exclusivement sur le Front-end. Cette application permettra à un utilisateur de visualiser les données météorologiques pour une ville donnée. L'utilisateur pourra spécifier la ville de son choix à l'aide d'un champ de saisie dédié. Les données pour la page météo seront présentées sous forme de texte avec des icônes afin de représenter chaque données (vents = icône représentant le vent, température = icône représentant la température). Les données pour la page statistique seront présentées sous forme de graphiques dynamiques et interactifs.
